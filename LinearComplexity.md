
# Linear Complexity: O(n)

Runtime tugas-tugas **linear** sangat bergantung pada input valuenya. Sebuah *function yg me-print array dengan 10 element* memiliki step lebih sedikit dibanding *function yang me-print array dengan 10k elemen*.

Perhatikan ini:

- function yg me-print array dengan 10 element = O(10)
- function yang me-print array dengan 10k elemen = 0 (10000)
